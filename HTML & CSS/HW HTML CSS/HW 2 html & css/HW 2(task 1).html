Активные вакансии

Наличие портфолио (проверяется при собеседовании); Знание языков: PHP, Javascript, Умение работать в PHP-фреймворках; Kohana/Yui/Laravel; Умение работать с СУБД MySQL; Умение писать веб-проекты с нуля по ТЗ; Большой опыт работы с 1CBitrix и другими распространенными CMS(WordPress, Joomla, MODx); Умение работать со связкой HTML+CSS+Javascript, опыт работы с Bootsrap; Приветствуется опыт работы с jQuery, AJAX, XML; Умение работать в команде; Умение разобраться в чужом коде(код наших проектов, как правило, достаточно грамотный).
